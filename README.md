\# SignalDesk



Multi-source regulatory and cybersecurity signal monitoring workspace.



\## Sources

\- BMJ (German legislation)

\- EDPB, EDPS (EU data protection)

\- ENISA (EU cybersecurity)

\- BSI (German cybersecurity)

\- BayLDA, LfD Berlin, Hessen (German state DPAs)

\- EUR-Lex (EU law)

\- AI Act (EU AI regulation)



\## Usage

```bash

python run\_scrapers.py all

python normalize\_all.py

```
